# 네이버시리즈 계정 관리자

## 실행 방법
1. 압축 해제
2. `index.html`을 브라우저에서 열기
3. 바로 사용

## 파일 구성
- `index.html`: 화면 구조
- `style.css`: PC 전용 스타일
- `app.js`: 데이터 로직, localStorage, 백업/복원

## 주요 특징
- 계정 화면을 카드형이 아닌 테이블형으로 구성
- 계정 행에서 직접 수정 가능
- 구매 작품은 계정별 상세 행에서 펼쳐서 관리
- 작품 리스트는 별도 표에서 검색/정렬/수정/삭제 가능
- 작품명은 datalist 자동완성 지원
- 데이터는 localStorage에 자동 저장
- JSON 백업/불러오기 지원

## 저장 위치
브라우저 `localStorage`

## 추천 사용
- Chrome 또는 Edge
- PC 해상도 1440px 이상
